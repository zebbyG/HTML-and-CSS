Introduction to the web and HTML.

The web

Is a collection of files that are made using the HTML language. Also, the web can be defined as HTML documents and other resources that refer to each other with links.

Web usually contain addresses http: or https: which stands for HyperText Transfer Protocol. This addresses are used to tell the server which rule/protocol to use.

HTML

HTML stands for Hypertext Markup Language.

Hypertext are texts that can have references (such as links) to other documents and resources.

Markup Are texts that have a special meaning.
These texts determine how a text in a web page looks like and how it's arranged.
Also, Markup can have references to include other documents like videos e.t.c

HTML is the main language used to write web pages.

Web Pages and Servers

- A browser gets web pages by communicating with the servers over the internet.
- Servers are computers that have running programs that give the browser its requests.

When programming using the HTML language, there are some concepts that should be followed:

  1. Syntax

  2. Formalism
    3. Computer do exactly what they are told to do. They do not correct any typos.
  3. Nesting

Markup

Every element starts and ends with a tag.
The first tag is known as the opening tag, and the one with a slash is called a closing tag.

HTML tags always start with the (<) sign and end with a (>) tag. These tags are knowns as angle brackets.

The words in between the tags, and the tags themselves is known as element.

Examples of tags are:

  • em
    •  Which means emphasize. It brings out words in italics.
  • br
    •  Which means break. Is a tag used to create a line break/skips to the next line. It is a void tag meaning it does not allow any closing tag.
  • strong
    •  Used to make the words bold.
  • h1
    •  It is a header tag used to write headers and runs from h1 (largest) to h6 (smallest). h1 is used for the main heading page, while the rest are used for sub-headings. Moreover, if the section comes after the other, we use the same level of heading.
  • mark
    •  Is used to highlight words.
  • p
    •  Which means paragraph. It separates a piece of text from other texts around it.
  • sub
    •  Meaning subscript, Is a tag used to position the words slightly below the other words.
  • sup
    •  Meaning superscript, Is a tag that places the words slightly above others.

Nested Elements

Elements on HTML can go inside other elements following a certain protocol. This means that the some elements must come before other elements when used together.

The top level elements always comes first, e.g, the paragraph elements and other big divisions. Inside these, there are other small division elements like the em elements.

These small division elements can also go inside other small division elements. For instance the mark element can go inside an em element and so on.

Block and Inline

Elements might be either block or inline

Block elements

As the name suggests, block elements are elements that adds a block/invisible box in between texts.

Block elements are used for large sections of text, such as paragraphs, headlines, or lists; and also for some other features such as video players and tables.
Examples of block elements are:
  • h1
  • p
  • div
and many more.

The div element

The div is a short for division. It is used to divide up the page into different sections.

It also has an invisible box around it which can also have a margin, a width, a height just like the (p) element.

Unlike the (p) element, the (div) element is a generic container for whatever other elements that might be put inside.

It is mainly used to organize the content and divide the page into sections.

Inline elements

These are the opposite of the block elements. They do not create blocks.
Some of the examples are:

  • br
  • em
  • mark
  • strong
and many more.

Lists and implied close tags

There are few HTML elements that are always used inside certain other elements, they are invalid on their own.
An example is the (li) element which is always nested in the (ol) tags or the (ul) tag.

The ul element

ul is an element used in listing items.
ul which means unordered lists, lists out items in the browser with a dot (.) before the name of the items.

The ol element

The ol element meaning odered lists is an element that displays items that are numbered unlike the ul elements that displays in dots.

Web Addresses

In the web standards, there are tecnical terms for these addresses.

A web addres is called a Uniform Resource Locater(URL)

  • UNIFORM = standerdized/consitent
  • RESOURCE = a file or other data object
  • LOCATOR = address for finding something

The URL is an address that tells the browser how to locate a certain resource like a file in a standerdized way.

There are different types of URLs, sometime they have the http/https some have the www.

Absolute URLs

Also known as a fully qualified URL. (https://www.example.com)

There are alot that can appear on URLs. For instance,

  1. https://en.wikipedia.org/
  2. file:///users/kelly/notes.html

The https simply tells the browser to use the secure encryption. In other cases it might be http which uses an insecure encryption to a web server.

The second URL is a URL for a file that is in the local computer. It starts with a file instead of the http/https.

The https,http and file are all protocols or ways that a browser can access something.

After the protocol, the : and the double // separates the protocol from the other part which is the domain name.
Domain name this tells the browser what server to connect to.

But the file URLs don't have the domain name because there is no server its just a file from your computer.

After the domain name the last thing that appers in a full URL is a file path, the name of the directory and the file the URL is referring to.
Sometimes it might be empty or a single slash if you are referring to the top page of a site.

Adding Images

The element for adding pictures in web browsers is the img tag. It works more like the link elements with some important differences.
Unlike the link element that has one attribute(href), the img uses two attributes:

  1. src
    2. This means source and is the URL of th image you want to appear in your webpage.
  2. alt
    3. Stands for alternate and its the text description of the image.

They both MUST be used when writing HTML.

The src attribute

The src attribute tells the browser where to get the image from.
The source of the image.

The alt element

This is the text that appears incase the image wasn't found in the source (src) or if it cannot display the image.

NB:Putting an img element in the HTML file doesn't copy the image, it tells the browser where to find that image file(source) and display it. The source might be your local disk or on a server. The img element is a void element. It is an inline element.

Documents: The DOCTYPE Tag

It is used to indicate what type of HTML is being used.
Unlike the other tags, it is written differently.
It is put at the top of every HTML document written.

The doctype tag tells the browser how to treat a HTML document(modern HTML) in this case.
It prevents bugs in complex HTMLs.
Without this tag browsers will go into quirks mode which is trying to be more compatible with older forms of HTML.

Documents: Head and Body

The Head

The head is where we put certain pieces of important information about the document. The title is the only required part of the head.
It tells the browser what title to put on a tab that displays the document. The title is written inside openning and closing tags of the title element.
The style sheets also appear in the head. This tells the browser what fonts and colors to display in the browser and how to position the text.
The head can also contain scripts which are programs that run in the browser. This is used in front-end web apps.

It also contains metadata which is the other information about the page, e.g, keywords for search engines.
The HTML tag can be added some characters which is the language your document is written in.(human language).
